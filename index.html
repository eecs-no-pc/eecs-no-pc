Note about gcc on mac. Xcode also installs a symlink so that you can run the g++ command on Mac. However, this is not g++! This invokes Apple Clang with a default set of arguments to emulate the real g++. The behavior of clang and gcc is different, so if a course asks you to use the gcc or g++ commands, do not run them on university-owned macs!

Windows Store Exploit

Not recommended.

ITS blocks the windows store on Sites PCs. However, it takes the windows store around 30 seconds after it launches to realize that it is blocked, and as long as you click the "install" button within that time frame, you can install anything on the windows store (like Python) without admin priviledges. If you fail this quick-time event, just log out or restart the PC and try again.

On CAEN Windows, the windows store is not blocked. You can use this to install WSL and Ubuntu.
