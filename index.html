C/C++ Guide Coursework in these classes is easily doable on Sites Macs, Sites Windows, Art Macs, CAEN Windows, and CAEN Linux. You get the following software relevant to the class, preconfigured for you to use, and can follow class setup tutorials 1:1: Groundworks Macs:
  • Xcode, Instruments Profiler & related tools
  • git
  • lldb
  • Apple Clang
  • "gcc" (notes about this later)
Note about gcc on mac. Xcode also installs a symlink so that you can run the g++ command on Mac. However, this is not g++! This invokes Apple Clang with a default set of arguments to emulate the real g++. The behavior of clang and gcc is different, so if a course asks you to use the gcc or g++ commands, do not run them on university-owned macs! Sites PCs:
