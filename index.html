EECS no PC Not all students have access to a personal device usable for running development tools. However, EECS courses do not always consider this problem. I was without a laptop for a few weeks during the second semester of my Junior year while it was being repaired, and it was stressful trying to figure out what resources were available to me while trying to meet tight assignment deadlines. I wrote this guide to help students who find themselves in a similar situation.

C/C++ Guide Coursework in these classes is easily doable on Sites Macs, Sites Windows, Art Macs, CAEN Windows, and CAEN Linux. You get the following software relevant to the class, preconfigured for you to use, and can follow class setup tutorials 1:1: Groundworks Macs:
  • Xcode, Instruments Profiler & related tools
  • git
  • lldb
  • Apple Clang
  • "gcc"*
Note about gcc on mac. Xcode also installs a symlink so that you can run the g++ command on Mac. However, this is not g++! This invokes Apple Clang with a default set of arguments to emulate the real g++. The behavior of clang and gcc is different, so if a course asks you to use the gcc or g++ commands, do not run them on university-owned macs!

Python Guide Python is preinstalled on macOS. However, you'll likely want an IDE or other editor to help you, especially with regard to pip libraries. PyCharm Community Edition is an excellent free python IDE which also manages python virtual environments and pip libraries for you. Below are the steps to run it without administrator priviledges:
    Any Mac
  1. Download the .dmg and open it
  2. Don't drag it to the Applications folder. Instead, just double-click it from the DMG window to open it.
    Any Windows device
  1. Download the .exe installer
  2. Use 7zip to unzip the installer .exe
  3. In the folder, navigate to bin/ and run pycharm64.exe.

ITS blocks the windows store on Sites PCs. However, it takes the windows store around 30 seconds after it launches to realize that it is blocked, and as long as you click the "install" button within that time frame, you can install anything on the windows store (like Python) without admin priviledges. If you fail this quick-time event, just log out or restart the PC and try again.
